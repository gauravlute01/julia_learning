### Julia Learning codes ###


#### Difference between Julia and Python ####
- In Julia, indentation and whitespace are not significant to syntax.
- Python index are 0 based; in julia, they are 1 based. 
- Julia does not support negative indexes. 
- Julia does not have an equivalent to the pass keyword in python. 
- Julia does not have a line-continuation syntax. 
- In julia, % is the remainder operator; in Python, it is the modulus operator.

#### Difference between Julia and R ####
- Julia's single quotes enclose characters, not strings. 
- In julia, strings can be created with triple quotes 
- In julia, modulus is mod(a, b), not a%%b
- The <-, <<-, and -> are not assignment operators in julia
- Julia does not treat the numbers 0 and 1 as Boolean values 
- In Julia, values are not copied when assigned or passed to a function
